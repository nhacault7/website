HTML & CSS: How Do They Make Websites Work?

There are many reasons for having your own website. From being a business owner to wanting to start a blog. Making your own can be a complex process so a great place to start is with the languages that make up a website.

HyperText Markup Language (HTML), is the most common language used for structuring webpages. Being a markup language, it differs from our preconceived notions of what coding is since HTML is more of a way to format text for web browsers.

At its core, HTML code is a collection of ‘elements’ that allows a web browser to segment sections of content. An element is made up of 4 components:

  1. An opening tag. (<p>)
  2. An attribute. (class=”name”)
  3. Your content. (I really love snakes)
  4. A closing tag. (</p>)

The opening and closing tags are found at the start and end of an HTML element. Opening and closing tags look almost identical except for the slash found in the closing tag. In the modern-day most tags carry semantic meaning, for accessibility reasons, so you have to choose the appropriate tag for the content inside. This content can range from an image, link or just plain text. Some elements may also include an attribute, which is just extra information useful to whoever is working on the code and is not visible on the document.

Here is an example of an HTML element that displays a paragraph:

<p class=”name”>I really love snakes</p>

Basic HTML flow

Every HTML document has a flow that dictates the order the elements are laid out. Individual elements can be one of two types.

Block-level: these elements take up the entire space of the line they are on regardless of the length of the content. This means that any element will start on a new line if it’s preceded by a block-level element in the code.

Inline: contrary to block-level elements, an inline element will only take up as much space as its content. You can compare this to the way a word document displays text where every word immediately follows the next until it reaches the end of the line or you hit the enter key.

What is CSS

Cascading Style Sheets (CSS), may not be something you are familiar with. However, it is responsible for styling web pages in the way we typically see them. Without CSS, every webpage would just be a long list of content with no colors, cool fonts, or even animations.

CSS allows you to target HTML elements with the use of a ruleset, which is made up of 3 parts:

  • The selector. (p)
  • A property. (color)
  • A value. (red)

The selector is responsible for targeting specific HTML elements by either their tag or identifying attributes. Once you’ve locked onto your specific element you can use a property to declare how you want to modify the element, followed by a value that determines the properties effect.

Here is an example of a CSS ruleset that changes a paragraph’s text color:

p {
 color: red;
}

Note that the property and value are followed by a colon and semicolon, and are also wrapped inside of curly braces.

There are many types of elements and rulesets but with this basic knowledge, you can start your journey to creating websites of your own! I would also highly recommend going to w3schools and reading their guide on both HTML and CSS
